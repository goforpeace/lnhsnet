# **App Name**: Landmark Estates

## Core Features:

- Hero Slider Image Management: Admin interface to upload, add, and delete image URLs for the hero slider.
- Project Creation: Admin interface to create a project with fields for title, description, flat sizes, land area, floors, parking, elevator, Google Maps location, status and images.
- Contact Form Submission Storage: Save contact form submissions to the Firebase real-time database.
- Project Display: Display projects in a responsive grid with title, description, status, and image slider. Dynamically loads content from the database.
- Dynamic Image Loading: Dynamically load hero image URLs from the real-time database for the full-width slider.
- Home Analytics: Dashboard page that shows total projects, inquiries, and other analytics.

## Style Guidelines:

- Primary color: Deep teal (#008080), conveying trust and sophistication, inspired by the brand's real estate focus.
- Background color: Light grey (#F0F0F0), offering a neutral backdrop that enhances readability and emphasizes the content.
- Accent color: Soft gold (#B8860B), used sparingly for highlights and CTAs, adding a touch of luxury and visual interest.
- Body font: 'PT Sans', a versatile sans-serif for readability and a modern look.
- Headline font: 'Playfair', a modern serif similar to Didot for headlines or small amounts of text, to give an elegant, fashionable, high-end feel.
- Clean, modern icons that represent each service (Developer, Property Management, Interior Design, IT Solutions).
- Centered layout with a max-width container of 1200px. Fully responsive using CSS grid/flex.
- Subtle fade and slide animations for the hero image slider. Smooth hover animations for the service cards.
